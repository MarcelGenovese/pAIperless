# pAIperless

**AI-Powered Document Processing Extension for Paperless-NGX**

pAIperless enhances your Paperless-NGX document management system with advanced AI capabilities, automated OCR processing, intelligent tagging, and action tracking through Google Calendar and Tasks integration.
